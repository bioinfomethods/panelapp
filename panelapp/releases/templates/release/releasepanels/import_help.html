Requirements
  • The import file must be in CSV format.
  • The first line of the file must contain the headers Panel ID and Promote.
  • Values for Panel ID must be natural numbers and refer to an existing panel.
  • Values for Promote must be either true or false.
Example

This is an example of a file that imports two panels; one to be promoted and the other not:

Panel ID,Promote
1,true
2,false
